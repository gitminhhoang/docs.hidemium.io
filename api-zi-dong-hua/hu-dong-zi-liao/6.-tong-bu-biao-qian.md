# 6.同步标签

## 申请

| 模式  | URL                                                                                                                           |
| --- | ----------------------------------------------------------------------------------------------------------------------------- |
| PUT | [http://127.0.0.1:5555/sync-tags](http://127.0.0.1:5555/sync-tags)[&#xD;](http://127.0.0.1:5555/profileList?page=1\&limit=10) |

**Body**

```
{
    "tags": ["1212121","bbbb1"],
    "profile_uuid": "992c1053-4ed6-4018-8776-f91a0a8da77b"
}
```

资料 uuid：资料的UUID

标签：资料标签限制为 15 个字符

## 响应

```
{
    "type": "success",
    "title": "Sync successfully.",
    "content": []
}
```
