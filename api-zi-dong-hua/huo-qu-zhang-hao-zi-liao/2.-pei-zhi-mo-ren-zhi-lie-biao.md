# 2. 配置默认值列表

## 申请

| 模式  | URL                                                         |
| --- | ----------------------------------------------------------- |
| GET | api\_url/get-list-config-default?page={page}\&limit={limit} |



| 参数    | 价值 | 描述      |
| ----- | -- | ------- |
| page  | 数字 | 分页的页数   |
| limit | 数字 | 显示的资料数量 |

## 申请示例

| 模式  | URL                                                                                                                              |
| --- | -------------------------------------------------------------------------------------------------------------------------------- |
| GET | [http://127.0.0.1:2222/get-list-config-default?page=1\&limit=10](http://127.0.0.1:5555/get-list-config-default?page=1\&limit=10) |

| 参数    | 价值 |
| ----- | -- |
| page  | 1  |
| limit | 10 |
