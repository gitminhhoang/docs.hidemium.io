# Change Proxy

Khi muốn thay đổi proxy cho nhiều profile cùng lúc, ta có thể chọn chức năng này. Đầu tiên bạn chọn các profile cần thêm proxy, sau khi chọn xong sẽ xuất hiện menu bên trên, bạn chọn proxy.
