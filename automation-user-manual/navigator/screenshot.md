| parameter                     | illustrate                                                                                        |
| ----------------------------- | ------------------------------------------------------------------------------------------------- |
| Screenshot name               | Bạn có thể nhập tên để lưu ảnh chụp màn hình hoặc để tên mặc định                                 |
| Save to folder: Default       | Tự động lưu ảnh vào folder Pictures                                                               |
