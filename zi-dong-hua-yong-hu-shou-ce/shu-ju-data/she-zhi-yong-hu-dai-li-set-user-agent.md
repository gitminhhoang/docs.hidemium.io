# 设置用户代理 (Set user agent)

<table><thead><tr><th width="258">参数</th><th>说明</th></tr></thead><tbody><tr><td>User agent</td><td>输入您要更改的用户代理</td></tr></tbody></table>

* 如图所示，有 3 个标签页，活动标签页是 3 号标签页
